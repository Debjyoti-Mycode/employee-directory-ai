# Employee Directory with AI Salary Assistant

## Features
- Add Employee
- Edit Employee
- Delete Employee
- Add Department
- Filter by Department
- Search by Name
- Sort by Name and Salary
- Dashboard Statistics
- AI Salary Suggestion using Gemini API

## Tech Stack
- Next.js App Router
- MongoDB Native Driver
- Tailwind CSS
- Google Gemini API

## Setup Instructions
1. Clone the repository
2. Run `npm install`
3. Create a `.env.local` file using `.env.example`
4. Run `npm run dev`

## Environment Variables
Use the values from `.env.example`

## Notes
Environment files are not committed for security reasons.
